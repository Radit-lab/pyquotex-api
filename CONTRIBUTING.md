# Contributing to PyQuotex API

Thank you for your interest in contributing!

## Important Notice

This is a modified version of the original [PyQuotex](https://github.com/cleitonleonel/pyquotex) library by **Cleiton Leonel Creton**. All contributions must comply with the **GNU GPL v3 license**.

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b feature/your-feature`
3. **Make your changes**
4. **Test your changes**: Ensure the API works correctly
5. **Commit with clear messages**: `git commit -m "Add: feature description"`
6. **Push to your fork**: `git push origin feature/your-feature`
7. **Open a Pull Request**

## Code Style

- Follow PEP 8 guidelines
- Add docstrings to functions and classes
- Keep code clean and minimal
- Add comments for complex logic

## Testing

Before submitting:
- Test API endpoints manually
- Ensure no breaking changes
- Verify Render deployment works

By contributing, you agree that your contributions will be licensed under the GNU GPL v3 license.

Open an issue for discussion before making major changes.
